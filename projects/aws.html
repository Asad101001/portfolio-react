AWS EC2 Deployment

Static Website & Cloud Infrastructure

Infrastructure Documentation

A foundational cloud hosting project aimed at hands-on deployment mechanics over native Linux server environments. Involved root-level manual provisioning bridging VPC networking, instance scaling limitations, and direct Nginx routing methodologies.

  • Manual EC2 Provisioning: Created and maintained an active `t3.micro` Ubuntu instance within the `eu-north-1` availability zone.
  • Protocol Restrictions: Designed precise Inbound/Outbound ICMP and TCP mapping rules mapping onto a configured AWS Security Group natively preserving server closure.
  • SSH Terminal Binding: Bound explicit root authorization certificates utilizing `.pem` private keys.
  • Nginx Routing: Bootstrapped raw web logic pipelines hosting static assets reliably via daemon routing mapping over default proxy limits.
